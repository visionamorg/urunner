# Epic: Strava Baseline Features
## Story: Gear and Shoe Mileage Tracker

**As a** serious runner,
**I want to** attach my specific running shoes to an activity,
**So that** the app tracks exactly how many kilometers I've put on each pair and warns me when they are "dead" (e.g., reaching 600km).

### Acceptance Criteria:
- *Given* I own multiple shoes, *when* I save a run, *then* I can select "Nike Alphafly" as the gear used.
- *Given* the Nike Alphaflys hit 500km total, *then* I receive an in-app warning that it's time to replace them to avoid injury.
