# Epic: Strava Baseline Features
## Story: Matched Runs (Historical Comparison)

**As a** runner who frequently runs the exact same loop,
**I want the** app to automatically group my "Matched Runs",
**So that** I can see a trend line of my pace improving over time on that specific route.

### Acceptance Criteria:
- *Given* I upload a 10k run around the local park, *when* the polyline matches an existing route I've run multiple times (90% similarity), *then* the activity page shows a "Matched Run" graph plotting my historical pace progress on this loop.
