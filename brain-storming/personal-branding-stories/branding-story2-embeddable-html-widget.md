# Epic: Personal Branding & Growth Features
## Story: Embeddable HTML Web Widgets

**As a** developer or blogger with my own personal website,
**I want to** copy a simple `<iframe src="...">` snippet from RunHub,
**So that** I can embed a living "Miles Run This Year" widget on my personal portfolio that automatically updates when I run.

### Acceptance Criteria:
- *Given* I am on the settings page, *when* I tap "Create Embedded Widget", *then* the system generates a tiny HTML snippet.
- *When* placed on a personal blog, *then* the iframe loads a beautifully styled mini-dashboard showing the user's YTD distance and the RunHub logo, driving free traffic to the platform.
