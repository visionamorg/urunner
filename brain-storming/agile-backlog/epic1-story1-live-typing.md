# Epic: Real-Time Social Engagement
## Story: Live Typing Indicators in Chat

**As a** community member in a chat room,
**I want to** see when other users are typing,
**So that** I know I am engaged in an active, real-time conversation.

### Acceptance Criteria:
- *Given* I am in a community room, *when* another user starts typing, *then* a "User is typing..." indicator appears.
- *Given* the user stops typing for 3 seconds, *then* the indicator disappears.
