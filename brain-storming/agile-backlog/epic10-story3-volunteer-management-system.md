# Epic: Event & Race Management
## Story: Volunteer Management System

**As a** community organizer,
**I want to** create volunteer slots (e.g., Water Station 1, Bag Drop) for an upcoming race,
**So that** community members can sign up to help, and I can coordinate event logistics seamlessly.

### Acceptance Criteria:
- *Given* I am viewing an event dashboard, *when* I click "Manage Volunteers", *then* I can create specific roles and capacities.
- *Given* a user cannot run but wants to participate, *when* they view the event page, *then* they can claim a volunteer slot. All volunteers are added to a dedicated, temporary chat room for coordination on race day.
