# Epic: Mobile Application (iOS & Android)
## Story: React Native Cross-Platform Architecture

**As a** development team,
**I want to** initialize the mobile codebase using React Native (Expo),
**So that** we can deploy to both iOS and Android simultaneously using a single JavaScript/TypeScript codebase.

### Acceptance Criteria:
- *Given* the repository is set up, *when* a developer runs the app, *then* it functions identically on the iOS Simulator and Android Emulator.
- *Then* the mobile app successfully consumes the existing Spring Boot REST APIs (`/api/auth`, `/api/activities`) using standard fetch/Axios implementations.
