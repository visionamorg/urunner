# Epic: Activity Canvas & Export Studio
## Story: "Year in Review Wrapped" Template

**As a** dedicated user at the end of the calendar year,
**I want to** select a special ultra-bold template that aggregates my entire year's worth of data into one graphic,
**So that** I can participate in the annual "Spotify Wrapped" style social media trend using RunHub.

**Status: Done**

### Acceptance Criteria:
- *Given* the user accesses the builder in December/January, *then* the "Annual Wrapped" template becomes prominently featured.
- *When* clicked, instead of showing a single run's data, the canvas queries the backend API to aggregate total kilometers run that year, total activities logged, and the user's most frequently run route.
- *Then* it applies highly vibrant, gradient-heavy, psychedelic background layers (similar to Spotify) behind massive bold text shouting "YOU RAN 1,200 KM THIS YEAR."
