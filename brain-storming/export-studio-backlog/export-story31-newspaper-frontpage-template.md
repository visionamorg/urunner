# Epic: Activity Canvas & Export Studio
## Story: "Newspaper Front Page" Template

**As a** user who just crushed a Personal Record (PR),
**I want to** export my run using a template styled like the front page of an old-school broadsheet newspaper,
**So that** the design acts as a humorous "BREAKING NEWS" announcement for my followers.

**Status: Done**

### Acceptance Criteria:
- *Given* I select the "News Headline" template, *then* the uploaded photo is rendered in High-Contrast Black & White using CSS `filter: grayscale(100%) contrast(120%)`.
- *Then* the primary typography utilizes a heavy, classic Serif font (like *Playfair Display* or *Times New Roman*).
- *Then* the `{title}` acts as the main headline (e.g., "LOCAL RUNNER CRUSHES 10K!"), with distance and pace listed as "sub-headlines" in column format.
