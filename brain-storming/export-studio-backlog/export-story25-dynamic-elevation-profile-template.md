# Epic: Activity Canvas & Export Studio
## Story: "Massive Elevation Profile" Template

**As an** ultra-marathoner or mountain runner,
**I want to** use a template where the elevation chart acts as the primary background art,
**So that** my followers can visually comprehend the massive 2,000m climb I just completed.

### Acceptance Criteria:
- *Given* a run contains elevation array data, *when* I select the "Elevation Profile" layout, *then* the canvas renders a massive, filled SVG area chart representing the altitude changes.
- *Then* this elevation chart anchors the bottom 50% of the screen, acting as a bold geometric silhouette against the background image.
