# Epic: Activity Canvas & Export Studio
## Story: "Retro 8-Bit Arcade" Template

**As a** gamer or nostalgist sharing a run,
**I want to** select a template that renders my run stats using classic 8-bit pixel art fonts and borders,
**So that** my activity export looks like a high-score screen from a 1980s arcade cabinet.

### Acceptance Criteria:
- *Given* I select the "8-Bit Arcade" template, *when* the canvas loads, *then* it applies a `Press Start 2P` or similar pixel-art web font to all text elements.
- *Then* the avatar frame is replaced with a blocky, aliased square border, and the `{distance}` takes the visual prominence of a "HIGH SCORE", colored in vibrant arcade yellow or pink against the photo.
