# Epic: Activity Canvas & Export Studio
## Story: "Negative Split Comparison" Template

**As a** data-driven runner who perfectly executed a race strategy,
**I want to** use a template that generates a beautiful bar-chart comparing the pace of my first half vs my second half,
**So that** I can boast about my "Negative Split" (running the end faster than the start).

### Acceptance Criteria:
- *Given* I select the "Pace Comparison" template, *then* the system calculates the average pace for the first 50% of the distance vs the second 50%.
- *Then* it renders a clean, visually striking side-by-side bar chart (e.g., "First Half: 5:30/km", "Second Half: 5:15/km") layered over the run photo.
