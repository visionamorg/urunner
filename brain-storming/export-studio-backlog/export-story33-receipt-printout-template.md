# Epic: Activity Canvas & Export Studio
## Story: "Thermal Grocery Receipt" Template

**As a** user who loves clever data visualizations,
**I want to** generate my run stats formatted to look exactly like an itemized CVS/grocery store thermal paper receipt,
**So that** the export is a quirky, highly shareable piece of typographic art.

**Status: Done**

### Acceptance Criteria:
- *Given* I select the "Receipt" template, *then* a long vertical white rectangle with zigzag/serrated top and bottom edges is centered on the canvas over the background photo.
- *Then* the text inside the bounds uses a thermal printer font (monospaced, slightly faded edge).
- *Then* the stats are "Itemized" (e.g., `Distance ...... 5.00 KM`, `Pace .......... 4:30 /KM`, `Tax ........... 0.00`, `TOTAL EFFORT .. MAX`).
