# Epic: Activity Canvas & Export Studio
## Story: "Neon Minimalist Route" Template

**As a** user running abstract or intricate city routes,
**I want to** use a template that hides all numbers and only shows my route line glowing like a neon sign on a pitch-black background,
**So that** the focus is entirely on the geographic "art" of the run.

### Acceptance Criteria:
- *Given* I select the "Neon Minimalist" template, *then* the background is forced to a solid color (e.g., #000000).
- *Then* the polyline SVG is rendered with a CSS `drop-shadow` giving it a highly aesthetic, glowing neon effect. All text stats (pace, distance) are hidden by default.
