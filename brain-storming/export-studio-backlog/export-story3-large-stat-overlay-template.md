# Epic: Activity Canvas & Export Studio
## Story: "Large Stat Overlay" Template

**As a** user who wants to boast about a massive distance,
**I want to** use a template dominating the screen with a single large number,
**So that** my followers instantly see my main achievement (e.g., a massive '21' for a half marathon).

### Acceptance Criteria:
- *Given* I select the "Large Stat" template, *then* the canvas places my total distance as a massive, abstract, and bold numeric overlay.
- *Then* the design includes geometric background shapes that intersect cleanly with the uploaded runner's photo.
