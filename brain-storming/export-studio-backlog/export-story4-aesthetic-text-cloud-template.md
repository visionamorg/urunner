# Epic: Activity Canvas & Export Studio
## Story: "Aesthetic Text & Cloud" Template

**As a** minimalist runner,
**I want to** use a template with very subtle stats and a customizable, bold title floating in the sky/cloud area,
**So that** the focus remains on the beautiful scenery of my run.

### Acceptance Criteria:
- *Given* I select the "Aesthetic Text" template, *then* my stats (profile/distance/time) apply as minimalist text at the very top of the canvas.
- *When* I tap the bold title text, *then* I can edit the string (e.g., from "MORNING RUN" to "SUNDAY LONG RUN"), rendering it in a thick, cloud-like aesthetic font.
