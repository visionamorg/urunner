# Epic: Activity Canvas & Export Studio
## Story: "Vintage Polaroid" Aesthetic Template

**As a** casual runner logging beautiful trail jogging moments,
**I want to** use a template that frames my photo like a classic instant Polaroid picture,
**So that** my export has a trendy, nostalgic, disposable-camera vibe.

### Acceptance Criteria:
- *Given* I select the "Polaroid" template, *then* my uploaded photo is cropped into a perfect square and surrounded by a thick white Polaroid-style border.
- *Then* the run stats (Date, Distance) are rendered in a custom handwritten/marker CSS font in the thick white margin below the photo.
