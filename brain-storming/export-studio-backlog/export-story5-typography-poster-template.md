# Epic: Activity Canvas & Export Studio
## Story: "Typography Poster" Template

**As an** artsy runner,
**I want to** use a bold, heavy brush-style typography template,
**So that** my export looks like an edgy, professional magazine graphic or race poster.

### Acceptance Criteria:
- *Given* I select the "Typography Poster" template, *then* the canvas uses heavy, sans-serif or brush-style fonts.
- *Then* the activity type (e.g., "5K RUN") is spelled out as a dominant design element integrated deeply with the background photo.

### Status: ✅ Done
**Implemented:** Typography Poster template with massive bold text ("5K RUN" style using distance + "RUN"), heavy 280px/160px sans-serif fonts, stats bar with pace/time/km, and username/date footer. Text shadow creates depth effect.
