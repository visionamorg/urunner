# User Stories

All feature requests and commands are tracked here as user stories.

## Structure
Each story lives in its own file named `US-XXX-short-title.md`.

## Status labels
- `[ ] Pending` — not started
- `[~] In Progress` — being implemented
- `[x] Done` — implemented and pushed
